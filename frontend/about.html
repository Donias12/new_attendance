About the System

Smart Attendance System

The Smart Attendance System is designed to streamline the attendance tracking process for educational institutions. Our platform provides an intuitive and efficient way for lecturers to track student attendance and for students to register their presence in classes.

Our Mission

Our mission is to simplify administrative tasks for educational institutions, allowing lecturers to focus on teaching and students to focus on learning, rather than spending time on manual attendance processes.

How It Works

Lecturers can create modules and generate unique session codes with custom expiration times. Students can then use these codes to mark their attendance. The system provides comprehensive reports and statistics for both students and lecturers.

Workflow

1

Lecturer Creates Module

Lecturers create modules with unique invite codes that students can use to join.

2

Students Join Module

Students register and join modules using the invite codes provided by lecturers.

3

Lecturer Generates Session Code

For each class session, lecturers generate a unique code with a custom expiration time.

4

Students Sign Attendance

Students use the session code to sign their attendance within the specified time period.

5

View Reports

Both lecturers and students can view comprehensive attendance reports and statistics.
